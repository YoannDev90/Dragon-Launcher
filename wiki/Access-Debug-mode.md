# How to Enable Debug Mode in Dragon Launcher

Dragon Launcher includes a hidden **Debug Mode** for advanced users and testers.  
Follow the steps below to enable it.

## Steps to Enable Debug Mode

1. Open **Dragon Launcher**.
2. Go to the **Settings** screen.
3. Scroll down to the **About** section.
4. Locate the **Version** label (the text showing the current app version, for example `2.5.3`).
5. Tap the **Version** name **8 times in a row**.
6. After the 8th tap, you will see a new menu entry showing up, that's the debug mode tab.

## Disabling Debug Mode

If you want to disable Debug Mode:

- Enter the debug mode tab
- Click the first toggle "Activate Debug Mode"
- Debug mode disabled instantly and returns you to the Settings screen

Debug Mode is intended for troubleshooting and advanced configuration.  
Only change options if you understand their impact.
